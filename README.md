# Sift

Sift는 문서 파일을 업로드하면 핵심 정보를 추출하고 요약해주는 AI 기반 문서 처리 서비스입니다.

단순한 문서 요약 도구를 넘어 문서를 다시 활용 가능한 형태로 정리·가공하는 것을 목표로 합니다.

---

## 1. 프로젝트 개요

- 문서 파일(PDF, Word 등)을 업로드하면 AI가 내용을 분석
- 핵심 내용을 요약해 사용자에게 제공
- 요약 결과를 다양한 방식으로 활용할 수 있도록 확장 예정

본 프로젝트는 **문서 요약 기능을 중심으로 한 서비스형 아키텍처**를 목표로 하며,  
프론트엔드와 백엔드를 분리하여 실제 배포 및 운영까지 진행될 예정입니다.

## 주요 기능 (1차 목표)

- 문서 파일 업로드
- 업로드된 문서의 텍스트 추출
- AI 기반 문서 요약 제공

---

## 향후 확장 계획

- 다양한 문서 형식 지원 (PDF, DOCX 등)
- 요약 결과 PDF / Word 형식으로 내보내기
- 문서 기반 질의응답 기능
- 요약 이력 및 버전 관리
- 사용자별 문서 관리 기능
